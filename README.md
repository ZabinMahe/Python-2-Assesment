# Read Every Assignment File Carefully and Solve Each Problem then Upload in Github Classroom

## 📌 How to Complete and Submit the Assignment

Follow the steps below to access, complete, and submit your assignment through GitHub Classroom:

### 🚀 Steps

1. Log in to your GitHub account in your browser.  
2. Copy and paste the GitHub Classroom assignment URL into the same browser.  
3. Click on the "Accept Assignment" button.  
4. Check your email and click on the given link (if required) to confirm the assignment again.  
5. Clone the repository (you can use GitHub Desktop — not available for Linux users — or use Git in terminal).  
6. Complete the assigned task in your local repository.  
7. Commit and push your changes to GitHub.  

➡️ Once you push your code to the repository, your assignment is considered **submitted automatically**.

---

### 💡 Notes
- Make sure you push your final work before the deadline.  
- You can update your submission anytime by pushing new commits before the deadline.  
- Always verify your repository to ensure your latest changes are uploaded.
